*Password Manager*
A simple command-line password manager written in Python. It allows you to generate strong passwords, save them to a text file, remove them, and view saved passwords.

+Concepts Used
Python standard libraries (random, string)
File handling (reading from and writing to text files)

+Features
Generate strong random passwords
Save passwords with a custom name
Remove a single saved password or all passwords
View a specific saved password or all saved passwords

+Requirements
Python 3.12.1 or any other stable Python 3 version
Place the Python script and the passwords file in the same folder
Do not change the name or extension of the passwords text file

+Important Notes (Usage Issues You May Face)
If you are using VS Code, enable the setting
Python › Terminal: Execute In File Dir so the script runs in the file’s directory.
On Windows, make sure Python has permission in Windows Security to modify files (especially if Controlled Folder Access is enabled).
